Converter
Приложение для конвертации валют с графическим интерфейсом, использованием внешнего API и сохранением истории операций.

Автор
Пономарев Григорий

Описание функционала
Конвертер валют предоставляет следующие возможности:

выбор валют для конвертации из расширенного списка (более 40 валют);
ввод суммы для конвертации;
получение актуальных курсов валют через внешний API;
отображение результата конвертации с указанием текущего курса;
сохранение истории последних 10 операций в таблице;
автоматическое сохранение истории в JSON‑файл;
загрузка истории при запуске приложения.
Технические требования
Python 3.6 или выше;
библиотеки: tkinter, requests, json, os.
Установка и запуск
Шаг 1. Установка зависимостей
Большинство библиотек уже встроены в Python. Установите только requests:

pip install requests

Шаг 2. Получение API-ключа
Перейдите на exhangerate-api.com
Зарегистрируйтесь и получите бесплатный API-ключ
Откройте файл converter.py и замените текущий ключ в строке: api_key = "YOUR_API_KEY" на ваш собственный API-ключ.
Шаг 3. Запуск приложения
Сохраните код в файл converter.py
Запустите приложение: python converter.py
Использование
Выберите валюту для конвертации в поле «Из валюты».
Выберите целевую валюту в поле «В валюту».
Введите сумму для конвертации в поле «Сумма».
Нажмите кнопку «Конвертировать».
Результат отобразится во всплывающем окне, а операция добавится в таблицу истории.
